# Modern Business Website — React Project

## Project Overview

This project is a multi-page responsive business website built using React 18.
It demonstrates modern frontend practices including reusable components, responsive layouts, accessibility considerations, and performance optimization.

## Features

* 5 Responsive Pages
* 15+ React Components
* Dark/Light Theme Toggle
* Portfolio Filtering
* Contact Form Validation
* Google Map Embed
* CSS Grid & Flexbox Layouts
* Animations & Transitions

## Technologies Used

* React (Hooks)
* JavaScript ES6+
* CSS3
* React Router
* HTML5 Semantic Structure

## Setup Instructions

### Install Dependencies

```bash
npm install
```

### Run Project

```bash
npm start
```

### Build Production Version

```bash
npm run build
```

## Folder Structure

* src/components → Reusable UI
* src/pages → Website pages
* src/styles → CSS system
* src/utils → Helper functions

## Accessibility Considerations

* Semantic HTML
* Color contrast
* Keyboard navigation support
* Responsive layouts
